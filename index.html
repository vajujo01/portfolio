Beerseker: The Stolen Brew

Top-Down Arcade action-comedy game with viking setting powered by Unreal Engine 5. Academic project for ESAT.

Tech: C++, Unreal Engine 5, Gameplay, Design, Team Work

Vulkan Graphics Engine

Graphics engine built with Vulkan (and OpenGL support), featuring PhysX 5 integration, a custom rendering system, Lua scripting, and an ECS architecture, limited version on Android Studio. Includes a custom editor powered by ImGui. Academic project for ESAT.

Tech: C++, Vulkan, OpenGL, PhysX 5, Lua Tech Blog

Project Night Club

A Unity-based game where you play as a bartender in different night clubs. The core gameplay involves crafting drinks while listening to the unique stories of your customers (Currently in Development). Personal project. All visual assets in this project were generated using AI tools. This is a non-commercial personal project.

Tech: C#, Unity

VR Mine Infinite Runner

Academic VR project developed in Unity with a partner. An infinite runner set in a mine where you shoot enemies and score points. Models & assets: Mixamo, Kenney, Unity Asset Store. Audio: YouTube Studio and other fonts. Non-commercial academic project.

Tech: C#, VR, Meta Quest 2/3, Unity

Unreal AI Behaviour

Academic project built in Unreal Engine 5. Features AI agents competing against each other using behavior trees. The demo includes: a patrolling agent, two agents fighting 1v1, and two teams in combat. Models and animations from Mixamo.

Tech: C++, Blueprints, Unreal Engine 5, AI

Fezio Galaxy

An academic video game developed in Unreal Engine 5 with a partner, in which the player takes on the role of an astronaut adrift in space. Using mechanics based on gravitational manipulation and perspective shifting, the player must solve a series of spatial puzzles to locate the missing components of their spacecraft and safely return home.

Tech: C++, Unreal Engine 5

Ping Pong VR

My Computer Engineering final thesis project. A VR Ping Pong experience developed in Unreal Engine 4, specifically optimized for Oculus Quest 2. Focused on realistic physics and tactile feedback.

Tech: C++, VR, Meta Quest 2, Blueprints, Unreal Engine 4

Funny Infinite Runner

Academic project developed in Unity with a partner. A funny infinite runner using Kenney assets and YouTube Studio audio tracks. Non-commercial academic project.

Tech: C#, Unity

Horror Mechanics Test

An Unreal Engine prototype dedicated to exploring interaction mechanics, atmospheric storytelling, and tension-building in the horror genre.

Tech: C++, Blueprints, Unreal Engine 4
